# PoemScript
My own markdown language for poems

### how to set up project :file_folder:

```
1) download this repo
```

### how to use :question:

```
python format.py [filename.txt]
```

#### standard syntax
```
// generates a <h1>-tag
[Title] 

// generates a <p>-tag in gray saying "by Author"
(Author) 

// generates a <p>-tag
text 
```
